Git naredbe

za pocetak ~ kretanje ~ dodavanje ~ gledanje ~ stapanje ~ deljenje

Za pocetak


    # da bi dohvatio udaljeni (remote) repo
    $git clone URL
    # ako postoji .gitmodules u repou, onda mora i
    $git submodule update --init --recursive

    # da bi napravio svoj
    $git init
    # pamti da napravis .gitignore
    # a, kad pozelis da podelis rad, moraces najpre
    $git remote add IME_REMOTA URL_REMOTA

    # git podrazumevano koristi "origin" kao ime remota

Za kretanje


    # da bi otisao gde god
    $git checkout COMMIT_ISH
    # commit-ish je sha1 id, grana, tag

    # napravi novu granu sa
    $git branch IME_NOVE_GRANE
    # obrisi granu sa
    $git branch -d IME_GRANE

    # napravi novi tag sa
    $git tag IME_NOVE_OZNAKE
    # obrisi tag sa
    $git tag -d IME_OZNAKE

Za dodavnje 1/2



    # najpre oznaci sta bi dodao
    $git add FILESPEC
    # ili dodaj sve
    $git add .

    # ako si pogresno oznacio, analogno
    $git reset FILESPEC
    $git reset .

Za dodavnje 2/2



    # potvrdi da je stanje ocekivano sa
    $git status

    # ako je sve u redu, sacuvaj novu verziju sa
    $git commit -v
    # prva linija je naslov, sto kraci
    # posle jedne prazne linije mozes da napises kratak opis

Za gledanje



    # pre svega: $git status
    # ali i
    $gitk --all &

Za stapanje



    # za stapanje grane u tekucu
    $git merge IME_GRANE

...ali ponekad dodje i do konflikta

Konflikt izgleda poput



    Bio jednom jedan zmaj

    <<<<<<< CURRENT BRANCH CONTENT
    A onda dosli seljani
    i roknuse zmaja :)

    @author KD
    =======
    I onda je zmaj sreo macu
    i bili su najboljii drugari
    >>>>>>> CHANGE WE TRY TO ADD TO IT

Konflikt se resava



    # kad zavrsis da prepravljas file u konfliktu,
    # oznacis ga kao resen sa:
    $git add IME_FILEA

    # kada resis i oznacis sve konflikte,
    # zavrsavas stapanje sa:
    $git commit -v
    # docekace te autogenerisana merge poruka :)

ako zelis da odustanes od resavanja konflikta, kucaj: $git merge --abort

Za deljenje



    # da bi objavio svoje izmene na udaljenom repou
    $git push

    # da bi dovukao izmene sa udaljenog repoa
    $git pull
