Malo terminologije

repo, radni dir, komitovi ~ git pokazivac ~ grane, oznake ~ lokal vs udaljen

Sta je git repozitorijum

  • git repo je bilo koji dir sa .git dirom u njemu.
  • radni dir (working dir) je sve osim .git dira
  • commit je zapakovana verzija koda cuvana u .git-u
  • komitovi su ulancani na svog prethodnika

Sta je git pokazivac

git pokazivac (commit-ish): SHA-1 id, grana, tag
...a moze biti i definisan i kao relativan!

Sta su onda grane

Oznake (tag) je najobicniji statican git POKAZIVAC

Grana (branch) je inkrementirajuci git POKAZIVAC

Na neki deo lanca KOMITOVA

Ostalo je: nasa estetska interpretacija git grafa.
...tako da grane bas i nisu grane.

ps. na tekucu granu pokazuje HEAD koji je opet samo git POKAZIVAC
