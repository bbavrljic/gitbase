git config ~ terminal ~ git help

Git minimalno podesavanja



    $git config --global user.name "TVOJE IME"
    $git config --global user.email "TVOJ@EMAIL"

    # samo za WINDOWS, a ako ne znas vim.
    $git config --global core.editor notepad

Terminal

...pored git naredbi trebace ti jos:



    # za pravljenje .filename datoteka poput .gitignore
    $touch .filename

    # ako zalutas u vim, pritisni
    <ESC>:q!<ENTER>

Git help



    # pozovi pomoc kucanjem --help iza bilo koje git naredbe
    $git init --help

    # kreci se za pola strane gore sa: u
    # kreci se za pola strane dole sa: d

    # trazi pomocu regex-a sa: /
    # ponovi pretragu nanize sa: n
    # ponovi pretragu navise sa: N

    # izlazak sa: q

ps. svaka naredba u terminalu ima pomoc ako ukucas --help iza nje
