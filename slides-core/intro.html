Zasto uposte git

zasto vcs ~ zasto git ~ dostupnost ~ cli vs gui

Zasto VCS

  • skalabilnost (copy/paste jednostavno nije dovoljan)
  • deljenje koda
  • vise aktualnih verzija istog projekta

Zasto git

  • jako je granularan
  • distriburan VCS, sto za sobom povlaci:
    • radi i offline
    • nemora da ima centralnu tacku autoriteta

Dostupnost alata

alat win linux mac
git x x x
gitk x x x*
kdiff3** x x x


* kada se instalira putem brew-a
** mora naknadno da se instalira i podesi u git configu
